# Security Policy

Helios is local-first software. The security model starts with user control over microphone access, transcripts, shell execution, and network behavior.

## Defaults

- No telemetry by default.
- No audio upload by default.
- No transcript upload by default.
- No shell execution unless explicitly enabled.
- No undeclared skill permissions.

## Permission Categories

- `audio`
- `desktop`
- `filesystem`
- `network`
- `notifications`
- `secrets`
- `shell`

## Reporting

Please report vulnerabilities privately to the maintainers with reproduction steps, affected version, and expected impact.
