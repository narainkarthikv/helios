# Go-To-Market Plan

## Ideal Users

- Linux developers who want local voice automation.
- Privacy-conscious power users.
- Small engineering teams managing Linux workstations.
- Open-source contributors building assistant skills.

## Wedge

Start with the local Linux agent:

- "Hey Helios, open terminal."
- "Hey Helios, summarize system status."
- "Hey Helios, run my standup script."
- "Hey Helios, start a timer."

The SaaS control plane becomes valuable when teams need managed skills, signed releases, device policy, audit logs, and fleet visibility.

## Launch Assets

- AppImage preview build.
- Demo video with hotword, voice input, voice output, and skill execution.
- Privacy whitepaper.
- Skill authoring guide.
- Public roadmap.
- Architecture decision records.

## Pricing Hypothesis

- Community: free local agent.
- Pro: managed sync, release channels, and private skill registry.
- Team: fleet policy, audit logs, SSO, and support.
